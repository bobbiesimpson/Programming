\section{size\textunderscore t}
\label{sec:size_t}

The type \verb=size_t= is used as the return type for the
\verb=sizeof= function in C. We might use it in the following way
\begin{lstlisting}
int n = 10;
size_t size = sizeof(int) * n;
int* p = malloc( size_t );
\end{lstlisting}
